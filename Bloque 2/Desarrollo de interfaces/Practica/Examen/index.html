Etiquetas HTML

Etiqueta Descripción
<table> </table> la Etiqueta table es una etiqueta que nos permite generar una table en el html, el contenido de esta etiqueta suele ir formado de <tr>, <td> o <th> las cuales explico más adelante, además es una etiqueta que siempre ha de ser cerrada
<tr> </tr> La Etiqueta tr es una etiqueta que nos permite generar una linea en la tabla, el contenido de esta etiqueta suele ir formado de <td> o <th>, además es una etiqueta que siempre ha de ser cerrada
<td> </td> La Etiqueta td es una etiqueta que nos permite generar columnas en la tabla, suele contener el texto en bruto de la tabla, además es una etiqueta que siempre ha de ser cerrada
<th> </th> La Etiqueta th es una etiqueta que nos permite generar en una linea la cabecera de una tabla en el html, su funcionalidad es la misma que td, pero con unos estilos predefinidos de texto centado y en negrita, además es una etiqueta que siempre ha de ser cerrada
<ol> </ol> La Etiqueta ol es una etiqueta que nos permite generar una lista ordenada en el html, el contenido de esta etiqueta suele ir formado de <li> la cual explicaré mas adelante, además es una etiqueta que siempre ha de ser cerrada
<ul> </ul> La Etiqueta ul es una etiqueta que nos permite generar una lista no ordenada en el html, el contenido de esta etiqueta suele ir formado de <li> la cual explicaré mas adelante, además es una etiqueta que siempre ha de ser cerrada
<li> </li> La Etiqueta li es una etiqueta que nos permite generar un elemento de las listas en <ol> o <ul>, nos añade un elemento mas al texto que suele ser una viñeta aunq podemos modificarla con css a gusto de cada uno, además es una etiqueta que siempre ha de ser cerrada
<div> </div> La Etiqueta div es una etiqueta que nos permite generar un contenedor en html,es el elemento más usado de html por las paginas ya que su manejabilidad es casi completa con el css, además es una etiqueta que siempre ha de ser cerrada
